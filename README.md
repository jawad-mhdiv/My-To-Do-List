# My-To-Do-List
A basic tack management application built for parcticing front-end development skills, allowing users to add and remove tasks easily.

## Features
- Add new tasks easily
- Mark tasks as completed
- Delete tasks
- Real-Time updates(no page reload)
- Responsive design for mobile and desktop
- Simple and user friendly UI

## Technology Used
- HTML5
- CSS3
- JavaScript
